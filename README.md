# Level Devil

**Level Devil** is an intense, notoriously difficult web-based platformer game that challenges players with unpredictable and frustrating levels. You never quite know what to trust! 

## Features
- Deceptive level design 
- Fully responsive web UI with a dark theme
- Built with HTML5 Canvas, TailwindCSS, and JavaScript

## Getting Started

To run the game locally, simply open the `public/index.html` file in your preferred web browser, or use a local development server like Live Server (for VSCode), Python's `http.server`, or Node's `http-server`.

### Example (using npx)
```bash
npx http-server ./public
```

Have fun and try to survive!
